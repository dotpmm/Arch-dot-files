# arch-rice-dot-files
My `hyprland` rice dotfiles

## Requires:
- `hyprland`
- `hyprpaper`
- `waybar`
- `wofi`
- `kitty`

To use these dotfiles, clone the repository and create symbolic links to the appropriate locations in your home directory. For example:

```bash
git clone
cd arch-rice-dot-files
ln -s $(pwd)/.config/hypr ~/.config/hypr
ln -s $(pwd)/.config/waybar ~/.config/waybar
ln -s $(pwd)/.config/wofi ~/.config/wofi
```

## Fonts used:
- JetBrainsMono Nerd Fonts

Feel free to customize the configurations to suit your preferences!

Make sure to install any additional dependencies that may be required for specific features or themes used in the configurations. Enjoy your customized Hyprland setup!
